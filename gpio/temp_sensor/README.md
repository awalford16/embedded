# Temperature Sensor

## Devices

Works with the Raspberry Pi Pico wired up to 5641AS 4-Digit LED display

### Pins

Pin | Description
---|---
0 | A
1 | B
2 | C (...)
8 | Digit 1
9 | Digit 2
10 | Digit 3
11 | Digit 4


## Build

Build with:

```
mkdir build
cd build

cmake ..
make

# Copy uf2 file across to pico
```
